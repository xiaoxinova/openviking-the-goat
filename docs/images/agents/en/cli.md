For developers who want to install, configure, and use OpenViking CLI directly from a local terminal.

### Step 1: Run the following command in your terminal to install OpenViking CLI and enter the configuration flow:

```bash
npm i -g @openviking/cli && ov config
```

### Step 2: When prompted, enter the OpenViking service URL and API Key. You can use the following service URL:

```text
https://api.vikingdb.cn-beijing.volces.com/openviking
```

### Step 3: After configuration, run the following command to view CLI usage:

```bash
ov --help
```
